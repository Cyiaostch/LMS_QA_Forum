Dependencies :
1. Python
2. Python Libraries : flask, pandas, numpy

Installation :
1. Install Python
2. Install Pip(python package manager)
3. Install libraries :
	command = "pip install flask pandas numpy"

Cara Pakai :
1. Buka folder dari terminal
2. Ketik command "python main.py"
3. Buka aplikasi di browser : "127.0.0.1:5000/home"
